PROJECTS

1st Year

IUS

Only one project. Objective of the project is to create Use case, ER diagram for information system

IZP

Huge project which will test basic knowledge of C language. Knowledge like memory allocation, pointers, structures and other basic building blocks of the language

IEL

Project is based on calculating few exercises where knoledge is gained during the study of this subject.

INC

Hardware subject, so the project is based on hardware language called VHDL. The project itself is not hard but people tend to dislike hardware languages in general.

IOS

2 projects. One in bash, which is usually for filtering data from files and second one is in C where you have to use fork for doing multiple tasks.

2nd Year

IPT

Project consists of multiple statistics exercises which will test knowledge gained during the study of the subject. Exercises range from easy to really hard ones.

ISS

One project which will usually be about signal processing. This one can be really fun.

INP

Another hardware subject which has 2 projects. FIrst one was in esoteric language called Brainf*ck and we had to do program for deciphering. Other project was in VHDL where we had to get our login printed.

IFJ

Really huge and complex group project where the aim is to create compiler which will work by the assignment.

IAL

2 projects. Aim of both is to implement multiple algorithm methods like methods over list, BVS or queues

IZG

Project which might be hard for some because it is about graphics and that is math and a lot of matrixes. Aim is to generate specific enviroment

IZU

4 small project in Prolog language. They are really short so one doesn't spend too much time on them. On are literally one liners.

IDS

Throughout this subject you work on project and submit it at 3 stages. First one is scheme, then basic database with basic filtering operations and then complex stuff like triggers, procedures etc.

IPK

Subject consists of 2 projects where you have to create client at first one and then server at second one. You are free to use C, C#, C++ or Common Lisp.

IPP

Whole subject is about objected oriented programming however the first project can be done without it. In the first one you have to create script for generating XML file for interpreter. Second project is much more harder because you have to implement said interpreter.

3rd Year

IIS

Another group complex project. You have to create information system, its frontend, backend and database. You are free to use any programming languages.

IMS

Only one project and can be done in group of 2. Aim is to model and simulate for example company and how they operate and find what they can cut or change to earn more money. It's done with SIMLIB library created in C.

IMP

Last of the hardware languages. One project that is connected to doing with some hardware equipment. One of the examples is create basic Snake-like game

ISA

Multiple different assignment done in C or C++. Some are easier than others.

ITU

You can heaviy reuse your project from IIS here because it's frontend what matters here.
